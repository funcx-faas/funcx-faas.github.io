Support

If you are using funcX or planning to try it, join the funcX Slack workspace.

If you would like to suggest new features, or participate in the discussion of implementation issues, please follow the project on GitHub.

We welcome contributions! If you would like to contribute to funcX via code, documentation, testing, or in any other manner, please read our contributing guide.
